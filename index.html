Bahmni

OpenSource Hospital Information System for low resource environments.

Source Repository
Mailing list

Bahmni project aims to build an open source hospital information system designed to be used in low resource environments. This system is envisioned to be built by integrating a multitude of open source applications such as OpenMRS, OpenERP, OpenELIS and JasperReports while building customizations and contributing back to the open source communities.
